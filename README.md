# Advanced CSS
Presentation to lecture about how the different features of CSS works.

### Short Agenda
######CSS Basics
---
* CSS Selectors
* CSS Variables
* CSS Units

######Block Styling
---
* Round borders
* Background attributes in CSS3
* Shadows and Gradients
* CSS Filters

######Transform/Animate
---
* Transformation with CSS in 2D...and 3D!
* Transition/Animation

######Layouts
---
* Flex and multicolumns
